## 1. Project Overview
The aim of this project is to design, develop, and launch a functional and user-friendly webshop that allows customers to browse products, add items to a shopping cart, and complete purchases online. The webshop will include secure payment integration, responsive design, and essential backend features like inventory management and order tracking.

## 2. Objectives
- Create a modern, responsive webshop
- Ensure smooth user experience (UX)
- Implement secure payment systems
- Build an easy-to-manage backend for product and order management
- Optimize for search engines (SEO)

## 3. Scope
**In scope:**
- Website design and development
- Product catalog and categories
- Shopping cart and checkout process
- Admin dashboard

## 5. Tools & Technologies
- **Frontend:** HTML, CSS, JavaScript, React/Vue (or other)
- **Hosting:** Netlify

## 8. Success Criteria
- Webshop goes live
- Fully functional
- Sales transactions processed successfully
